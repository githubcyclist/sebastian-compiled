Sebastian

Sebastian is a program hosting the best virtual friend you will ever have – because not only is he your friend – he is also a virtual assistant!


What is Sebastian?

Sebastian is a virtual assistant program which hosts the character of your choice.

The default character is Peedy the parrot. Download more characters

here.

Note that on Windows Vista and later, a lot of these characters will not install without an extra step.
Right-click on the character installer of your choice, and select "Run as administrator".
You will be prompted. Click Yes. If it asks for a password, contact your computer administrator.
